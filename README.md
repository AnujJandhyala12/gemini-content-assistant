# Gemini AI Content Assistant & Developer Tools

A multi-purpose Python application featuring an AI assistant built using the official Google GenAI SDK. This project features structured email generation, code analysis tools, and a dynamic developer playground wrapped up in an interactive web application interface.

## 🚀 Features

* **Feature 1: Basic Chat** - Fast interaction with Gemini models exploring foundational computing definitions and analogies.
* **Feature 2: Smart Email Writer** - High-utility corporate assistant executing context-aware emails using structured parameters (recipient, tone, purpose, key bullets).
* **Feature 3: Code Explainer & Debugger** - Elite developer tool built to dissect logical flows, explain functionality step-by-step, catch compiler/runtime bugs, and supply fully refactored scripts.

---

## 🛠️ Tech Stack

* **Language:** Python 3.11
* **AI Engine:** Google GenAI SDK (`gemini-2.5-flash`)
* **Web Framework:** Streamlit
* **Containerization:** Docker
